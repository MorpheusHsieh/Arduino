  (5).查詢目前工作模式：
      
      ESP-01S有三種工作模式，分別是：
      1 -> Station (STA) 網卡模式
      2 -> Access Point (AP) 基地台模式
      3 -> BOTH (AP + STA).
      
      注意, 執行 CWMODE 指令指定工作模式後，最好立即執行 CWRST 重啟, 以確認設定生效。
      
      AT+CWMODE?
      +CWMODE:1
      
      OK   

  (6).設定工作模式：
  
      AT+CWMODE=1

      OK  

  (7).查詢IP：

      OK

      從上述訊息可知此模組已經被指派 "192.168.4.2" 的 IP 了。

  (11).建立 TCP/IP 連線：
  
       AT+CIPSTART="TCP","192.168.4.1",80
      CONNECT
      
      OK

  (12).查詢目前 TCP/IP 連線狀態：
  
      AT+CIPSTATUS
      STATUS:3
      +CIPSTATUS:0,"TCP","192.168.4.1",80,35250,0
      OK 

  (13).使用 GET 指令取得 HTTP Server 的訊息：
  
      此部分指令分為兩部分：
      
      A.告訴伺服器要傳送的指令有幾個位元：
      
        AT+CIPSEND=7
      
      B.在 '>' 符號出現後，輸入 GET 指令：
      
        GET /
        
        傳送的的資料有 5個位元，再加上 '\r\n' 剛好是 7個位元。
        
        將會收到從 HTTP Server 回傳的資料。
        
      執行過程如下：
      
        AT+CIPSTART="TCP","192.168.4.1",80
        
        CONNECT
        
        OK
        AT+CIPSEND=7
        
        
        OK
        > 
        Recv 7 bytes
        
        SEND OK
        
        +IPD,211:<html><head>
        <meta http-equiv="refresh" content="10">
        <title>From ESP-01</title>
        </head><body>
        <p>ClientMsg: ,7:GET /
        </p>
        <p>Humidity: 53.00%</p>
        <p>Temperature: 21.00*C</p>
        </body>
        </html>
        CLOSED
      
      PS: 我已事先在伺服器建立了一個溫濕度感測的網頁，
      
          所以回傳回來的資訊就是網頁內容。
  
  (14).關閉 TCP/IP 連線：
  
      AT+CIPCLOSE

      ERROR

      這是因為只有在 TCP/UDP CONNECTING 或 CONNECT OK 狀態下，
      
      才會關閉連線，否則會認為關閉失敗傳回 ERROR。
